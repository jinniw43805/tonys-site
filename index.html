Coconut

The Note Platform

Create and Organize. Take collaborative notes, share data and files, and use comments to share your thoughts in real-time or asynchronously.
Everyone taking the course can edit class note,and others would see the change in the same time.

Show the courses you taking and find more interesting courses between those your friends taking!
Fit all mobile device so you can use this service anywhere anytime through your phone or pad.

The note core is using firepad ,an open source collaborative code and text editing

Firepad can use either the CodeMirror editor or the Ace editor to render documents, and its operational transform code borrows from ot.js. Behind the scenes, Firepad uses Firebase for cloud data storage and synchronization.
